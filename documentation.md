EcoKigali

My Frontend CAT Project

This is a web application I built to help people in Kigali report environmental problems like illegal dumping, missed waste collection, and pollution.

The website is made for the three districts: Gasabo, Kicukiro, and Nyarugenge.

What you can do on the website:
- Report any environmental issue (with form validation)
- See live reports and map of Kigali
- Check upcoming Umuganda and clean-up events
- View Facebook and Twitter feeds
- Read about real green initiatives in Kigali

Technologies I used:
- HTML5, CSS3 and Bootstrap 5
- JavaScript
- Leaflet.js (for the map)

How to run it:
1. Open the folder
2. Double-click index.html
3. The website will open in your browser
